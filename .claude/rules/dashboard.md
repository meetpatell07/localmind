# Rules: Dashboard

- Next.js 14 App Router, RSC + streaming
- Tailwind + shadcn/ui, Lucide icons
- State: React hooks + Zustand (no Redux)
- Forms: React Hook Form + Zod
- Tabs: Chat | Memory | Planner | Files | Voice
- Dark mode default, system preference
- Stream LLM tokens as they arrive (ReadableStream → React state)
- Markdown: react-markdown + remark-gfm
- Loading: skeleton components, never blank
- Errors: friendly message + retry + log to error system
- Mobile responsive (for remote access)
- No auth for sprint 1 (localhost only, single user)
